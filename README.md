# Deep Down Trader

Small browser game prototype for GitHub Pages.

## Run locally

Open `index.html` in a browser.

## Publish on GitHub Pages

1. Create a GitHub repository and upload `index.html`, `styles.css`, and `game.js`.
2. In the repository settings, open the Pages section.
3. Set the source to deploy from your main branch.
4. GitHub Pages will publish the site at `https://your-username.github.io/repository-name/`.
